# Business Operating System Constitution

Status: Approved foundation rule
Owner: Saeed
Applies to: ChatGPT, Claude, and all project files

## Mission
Build a practical business operating system for Corella, the Gamla Stan project, and future hospitality projects.

## Main Rule
The system must be useful for one owner-operator. Simple, clear, and executable is better than complex and theoretical.

## Roles
Saeed: final decision maker and approver.
ChatGPT: system architect, structure, standards, integration.
Claude: research, review, critique, and second analysis.
GitHub: single source of truth.

## Change Rule
No AI may create, edit, commit, delete, rename, or push files without explicit approval from Saeed.

## Document Status
Draft: not final.
Review: ready for critique.
Approved: official working version.
Experimental: idea only, not decision material.

## File Rule
Each file should have one clear purpose. Avoid large mixed documents.

## Practicality Rule
Every recommendation should be executable, measurable, or useful for decision-making.
