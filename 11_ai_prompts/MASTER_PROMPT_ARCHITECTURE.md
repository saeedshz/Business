# Master Prompt Architecture

## Goal
Create practical, measurable and reusable outputs for hospitality projects.

## Required Layers
1. Permanent principles
2. Current business context
3. Site-specific brief
4. AI review rules

## Output Rule
Do not only give advice. Create usable assets such as checklists, SOPs, tables and decision tools.

## Practicality Rule
Prefer the simplest executable option over a complex theoretical option.

## Review Rule
Before finalizing, list weak points, missing data and next actions.
