# Business Operating System

This repository contains the evolving Business Operating System (BOS)
for hospitality projects.

Structure:
- 00_project_constitution: Core principles
- 01_business_context: Permanent business context
- 02_projects: Individual projects
- 03_sop_library: Standard Operating Procedures
- 04_templates: Reusable templates
- 05_ai_outputs: AI-generated analyses
